## 🚀 Featured Projects

<details>
<summary><b>🔬 RISC-V RAG Pipeline for RTL Generation</b> — <i>83–100% benchmark pass rate</i></summary>

<br/>

> **Stack:** Python · LangChain · ChromaDB · Groq (Llama-3.3-70b) · FastAPI · Verilator · CI/CD

- Production RAG pipeline that generates synthesizable RISC-V Verilog RTL from natural language descriptions
- Agentic self-correction loop: Verilator lint errors are fed back to the LLM — avg. **1.2–1.8 attempts** to correct
- Hardware-aware chunking (module-boundary splits), Top-K=5 dense retrieval
- Validated on RV32I single-cycle core — **11/11 functional testbench assertions passed** (ALU, memory, branch, jump)
- Offline embedding caching, structured logging, network-resilient corpus construction
